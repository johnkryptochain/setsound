# SetSound

SetSound est une application web d'édition et de traitement audio. Elle permet de couper, joindre, convertir, enregistrer et compresser des fichiers audio, ainsi que de détecter le BPM.

## Open Source

Ce projet est **open source et entièrement libre**, développé par **Jema Technology**.
